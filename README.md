# blender_lib

## install

```
git submodule add git@github.com:BloomScheme/blender_lib.git ./blender_lib
```

## setup

```bash
git submodule update --init --recursive
```

## update

```
git submodule update -i
```

## remove

```bash
git submodule deinit -f ./blender_lib
git rm -f ./blender_lib
```
